## Manager Block Ultra Pro

Plugin premium de bloques personalizados Gutenberg con ACF y panel de configuración.

## Características
- Bloques Gutenberg avanzados con ACF.
- Carga condicional de assets.
- Panel de administración.
- Soporte para futuras actualizaciones automáticas.
- Código orientado a objetos limpio y modular.

## Instalación
1. Sube la carpeta `manager-block-ultra-pro` a `wp-content/plugins/`.
2. Activa el plugin desde el panel de WordPress.

## Bloques
- Titulo y Subtitulo.
- Carousel de imagenes con textos `los textos se quedan siempre fijos, lo unico se me desplaza son las imagenes`.
- Imagen de fondo con caja de información.
